新約聖經經文彙編

腓立比書第2章信徒榜樣

主要內容包括

（１）國語和合本

（２）中文新譯本

（３）當代中譯本

（４）英文欽定本（KJV）

（５）英文國際本（NIV）

 

中/英文經文彙編

【當以基督的心為心】

腓 2：1--國語和合本

　所以在基督裏若有甚麼勸勉、愛心有甚麼安慰、聖靈有甚麼交通、心中有甚麼慈悲憐憫、

 

腓 2：2--國語和合本

　你們就要意念相同、愛心相同、有一樣的心思、有一樣的意念、使我的喜樂可以滿足。

 

腓 2：3--國語和合本

　凡事不可結黨、不可貪圖虛浮的榮耀．只要存心謙卑、各人看別人比自己強。

 

腓 2：4--國語和合本

　各人不要單顧自己的事、也要顧別人的事。

 

腓 2：5--國語和合本

　你們當以基督耶穌的心為心。

 

腓 2：6--國語和合本

　他本有　神的形像、不以自己與　神同等為強奪的．

 

腓 2：7--國語和合本

　反倒虛己、取了奴僕的形像、成為人的樣式．

 

腓 2：8--國語和合本

　既有人的樣子、就自己卑微、存心順服、以至於死、且死在十字架上。

 

【無不口稱耶穌為主】

腓 2：9--國語和合本

　所以　神將他升為至高、又賜給他那超乎萬名之上的名、

 

腓 2：10--國語和合本

　叫一切在天上的、地上的、和地底下的、因耶穌的名、無不屈膝、

 

腓 2：11--國語和合本

　無不口稱耶穌基督為主、使榮耀歸與父　神。◎

 

腓 2：12--國語和合本

　這樣看來、我親愛的弟兄、你們既是常順服的、不但我在你們那裏、就是我如今不在你們那裏、更是順服的、就當恐懼戰兢、作成你們得救的工夫．

 

腓 2：13--國語和合本

　因為你們立志行事、都是　神在你們心裏運行、為要成就他的美意。

 

【門徒如明光照耀】

腓 2：14--國語和合本

　凡所行的、都不要發怨言、起爭論、

 

腓 2：15--國語和合本

　使你們無可指摘、誠實無偽、在這彎曲悖謬的世代、作　神無瑕疪的兒女．你們顯在這世代中、好像明光照耀、

 

腓 2：16--國語和合本

　將生命的道表明出來、叫我在基督的日子、好誇我沒有空跑、也沒有徒勞。

 

腓 2：17--國語和合本

　我以你們的信心為供獻的祭物．我若被澆奠在其上、也是喜樂、並且與你們眾人一同喜樂．

 

腓 2：18--國語和合本

　你們也要照樣喜樂、並且與我一同喜樂。

 

【保羅稱讚提摩太】

腓 2：19--國語和合本

　我靠主耶穌指望快打發提摩太去見你們、叫我知道你們的事、心裏就得著安慰。

 

腓 2：20--國語和合本

　因為我沒有別人與我同心、實在罣念你們的事。

 

腓 2：21--國語和合本

　別人都求自己的事、並不求耶穌基督的事。

 

腓 2：22--國語和合本

　但你們知道提摩太的明證、他興旺福音與我同勞、待我像兒子待父親一樣。

 

腓 2：23--國語和合本

　所以我一看出我的事要怎樣了結、就盼望立刻打發他去．

 

腓 2：24--國語和合本

　但我靠著主自信我也必快去。

 

腓 2：25--國語和合本

　然而我想必須打發以巴弗提到你們那裏去．他是我的兄弟、與我一同作工、一同當兵、是你們所差遣的、也是供給我需用的。

 

腓 2：26--國語和合本

　他很想念你們眾人、並且極其難過、因為你們聽見他病了．

 

腓 2：27--國語和合本

　他實在是病了、幾乎要死．然而　神憐恤他、不但憐恤他、也憐恤我、免得我憂上加憂。

 

腓 2：28--國語和合本

　所以我越發急速打發他去、叫你們再見他、就可以喜樂、我也可以少些憂愁。

 

腓 2：29--國語和合本

　故此你們要在主裏歡歡樂樂的接待他．而且要尊重這樣的人．

 

腓 2：30--國語和合本

　因他為作基督的工夫、幾乎至死、不顧性命、要補足你們供給我的不及之處。

 

資料來源

（１）和合本聖經經文和中文新譯本聖經由本人打字輸入電腦並經校對
