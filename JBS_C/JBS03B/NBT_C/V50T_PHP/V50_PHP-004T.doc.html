中/英文經文彙編

腓 4：1--國語和合本

　我所親愛所想念的弟兄們、你們就是我的喜樂、我的冠冕．我親愛的弟兄、你們應當靠主站立得穩。◎

 

腓 4：2--國語和合本

　我勸友阿爹和循都基、要在主裏同心．

 

腓 4：3--國語和合本

　我也求你這真實同負一軛的、幫助這兩個女人、因為他們在福音上曾與我一同勞苦．還有革利免、並其餘和我一同作工的．他們的名字都在生命冊上。

 

【門徒當在主裏喜樂】

腓 4：4--國語和合本

　你們要靠主常常喜樂．我再說、你們要喜樂。

 

腓 4：5--國語和合本

　當叫眾人知道你們謙讓的心。主已經近了。

 

腓 4：6--國語和合本

　應當一無罣慮、只要凡事藉著禱告、祈求、和感謝、將你們所要的告訴　神。

 

腓 4：7--國語和合本

　　神所賜出人意外的平安、必在基耶穌裏、保守你們的心懷意念。

 

【凡可稱讚的都要思念】

腓 4：8--國語和合本

　弟兄們、我還有未盡的話．凡是真實的、可敬的、公義的、清潔的、可愛的、有美名的．若有甚麼德行、若有甚麼稱讚、這些事你們都要思念。

 

腓 4：9--國語和合本

　你們在我身上所學習的、所領受的、所聽見的、所看見的、這些事你們都要去行．賜平安的　神、就必與你們同在。

 

【無論甚麼景況都可知足】

腓 4：10--國語和合本

　我靠主大大的喜樂、因為你們思念我的心、如今又發生．你們向來就思念我、只是沒得機會。

 

腓 4：11--國語和合本

　我並不是因缺乏說這話、我無論在甚麼景況、都可以知足、這是我已經學會了．

 

腓 4：12--國語和合本

　我知道怎樣處卑賤、也知道怎樣處豐富、或飽足、或飢餓、或有餘、或缺乏、隨事隨在、我都得了祕訣。

 

腓 4：13--國語和合本

　我靠著那加給我力量的、凡事都能作。

 

腓 4：14--國語和合本

　然而你們和我同受患難、原是美事。

 

腓 4：15--國語和合本

　腓立比人哪、你們也知道我初傳福音、離了馬其頓的時候、論到授受的事、除了你們以外、並沒有別的教會供給我．

 

腓 4：16--國語和合本

　就是我在帖撒羅尼迦、你們也一次兩次的、打發人供給我的需用。

 

腓 4：17--國語和合本

　我並不求甚麼餽送、所求的就是你們的果子漸漸增多、歸在你們的賬上。

 

腓 4：18--國語和合本

　但我樣樣都有、並且有餘．我已經充足、因我從以巴弗提受了你們的餽送、當作極美的香氣、為　神所收納、所喜悅的祭物。

 

腓 4：19--國語和合本

　我的　神必照他榮耀的豐富、在基督耶穌裏、使你們一切所需用的都充足。

 

腓 4：20--國語和合本

　願榮耀歸給我們的父　神、直到永永遠遠。阿門。◎

 

腓 4：21--國語和合本

　請問在基督耶穌裏的各位聖徒安。在我這裏的眾弟兄都問你們安。

 

腓 4：22--國語和合本

　眾聖徒都問你們安。在該撒家裏的人特特地問你們安。◎

 

腓 4：23--國語和合本

　願主耶穌基督的恩常在你們心裏。
