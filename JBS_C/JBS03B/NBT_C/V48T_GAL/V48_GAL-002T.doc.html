加拉太書 第2章 福音真理

中/英文經文彙編

保羅巴拿巴奉啟示再往耶路撒冷（加 2：1-8 CUV）

加 2：1--國語和合本（CUV）

　過了十四年、我同巴拿巴又上耶路撒冷去、並帶者提多同去。

 

加 2：2--國語和合本（CUV）

　我是奉啟示上去的、把我在外邦人中所傳的福音、對弟兄們陳說．卻是背地裏對那有名望之人說的．惟恐我現在、或是從前、徒然奔跑。

 

加 2：3--國語和合本（CUV）

　但與我同去的提多、雖是希利尼人、也沒有勉強他受割禮．

 

加 2：4--國語和合本（CUV）

　因為有偷者引進來的假弟兄、私下窺探我們在基耶穌裏的自由、要叫我們作奴僕．

 

加 2：5--國語和合本（CUV）

　我們就是一刻的工夫、也沒有容讓順服他們、為要叫福音的真理仍存在你們中間。

 

加 2：6--國語和合本（CUV）

　至於那些有名望的、不論他是何等人、都與我無干．　神不以外貌取人．那些有名望的、並沒有加增我甚麼．

 

加 2：7--國語和合本（CUV）

　反倒看見了主託我傳福音給那未受割禮的人、正如託彼得傳福音給那受割禮的人．

 

加 2：8--國語和合本（CUV）

　（那感動彼得、叫他為受割禮之人作使徒的、也感動我、叫我為外邦人作使徒）

 

雅各磯法約翰向保羅巴拿巴行相交之禮（加 2：9-10 CUV）

加 2：9--國語和合本（CUV）

　又知道所賜給我的恩典、那稱為教會柱石的雅各、磯法、約翰、就向我和巴拿巴用右手行相交之禮、叫我們往外邦人那裏去、他們往受割禮的人那裏去．

 

加 2：10--國語和合本（CUV）

　只是願意我們記念窮人．這也是我本來熱心去行的。

 

保羅責備磯法與外邦人隔開（加 2：11-19 CUV）

加 2：11--國語和合本（CUV）

　後來磯法到了安提阿、因他有可責之處、我就當面抵擋他。

 

加 2：12--國語和合本（CUV）

　從雅各那裏來的人、未到以先、他和外邦人一同喫飯．及至他們來到、他因怕奉割禮的人、就退去與外邦人隔開了。

 

加 2：13--國語和合本（CUV）

　其餘的猶太人、也都隨者他裝假．甚至連巴拿巴也隨夥裝假。

 

加 2：14--國語和合本（CUV）

　但我一看見他們行的不正、與福音的真理不合、就在人面前對磯法說、你是猶太人、若隨外邦人行事、不隨猶太人行事、怎麼還勉強外邦人隨猶太人呢。

 

加 2：15--國語和合本（CUV）

　我們這生來的猶太人、不是外邦的罪人、

 

加 2：16--國語和合本（CUV）

　既知道人稱義、不是因行律法、乃是因信耶穌基、連我們也信了基耶穌、使我們因信基稱義、不因行律法稱義．因為凡有血氣的、沒有一人因行律法稱義。

 

加 2：17--國語和合本（CUV）

　我們若求在基裏稱義、卻仍舊是罪人、難道基是叫人犯罪的麼．斷乎不是。

 

加 2：18--國語和合本（CUV）

　我素來所拆毀的、若重新建造、這就證明自己是犯罪的人。

 

加 2：19--國語和合本（CUV）

　我因律法、就向律法死了、叫我可以向　神活者。

 

基督在保羅裏面活者（加 2：20-21 CUV）

加 2：20--國語和合本（CUV）

　我已經與基督同釘十字架．現在活者的、不再是我、乃是基督在我裏面活者．並且我如今在肉身活者、是因信　神的兒子而活、他是愛我、為我捨己。

 

加 2：21--國語和合本（CUV）

　我不廢掉　神的恩．義若是藉者律法得的、基督就是徒然死了。
